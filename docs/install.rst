
Installation
==============

Required Dependencies
-----------------------
 - Drupal:

  - Drupal 8.8.x or 9.0.x (8.8.x recommended)
  - Drupal core modules: Search, Path, View, Entity, and PHP modules.

- PostgreSQL
- PHP 7.3+
- UNIX/Linux

Specific Installation Guides
-------------------------------

The recommended method to get involved in Tripal 4 development is the native docker installation. This most closely mimics the enviornment Travis CI is run on.
